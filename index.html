Injectplate Component

Getting Started

You can either download a copy of the source files or install Injectplate via Bower.

bower install injectplate

Start by including the necessary files.

<body>
    /* Your content goes here */
    <script src="js/injectplate.min.js"></script>
</body>

Getting Started With NPM

If you instead wish to use Injectplate as a Node module simply install using the following command.

npm install injectplate

What Is This?

Injectplate is a declare once Javascript component injector. This allows you to create HTML components and inject them into the DOM at will. In essence it is a wrapper for the awesome Mustache.js library which has a great template sytax.


Basic Example

Initialize Injectplate before creating your first component.

<script>
// Initialize
var Inject = Injectplate.init();

// Create component
Inject.component({
    name: 'article',
    className: 'basic-article',
    html: `
        <article>
            <h2>{{heading}}</h2>
            <div>{{content}}</div>
        </article>
    `
});
</script>

Once the component has been created, simply bind it to an element or generate the HTML and parse in the relevant data.

<script>
// Bind component
Inject.bind({
    component: 'article',
    to: '#article',
    data: {
        heading: 'Great Article Heading',
        content: 'This will just be some basic text about stuff.'
    }
});
// Generate component HTML
var myComponent = Inject.generate({
    component: 'article',
    data: {
        heading: 'Great Article Heading',
        content: 'This will just be some basic text about stuff.'
    }
});
</script>

Dynamic Data

Sometimes you might require elements to be generated dynamically and Injectplate can do this with a JSON based dataset. To do so modify the HTML option of the component. NOTE that the HTML template is defined via an array but can be a basic string.

To see how to define templates view the online documentation. Find an example below:

<script>
Inject.component({
    name: 'articleAdvanced',
    className: 'advanced-article',
    html: `
        <article>
            <h2>{{heading}}</h2>
            <div class="content">
                {{#paragraphs}}
                    <p>{{content}}</p>
                {{/paragraphs}}
            </div>
        </article>
    `
});

Inject.bind({
    component: 'articleAdvanced',
    to: '#article',
    data: {
        heading: 'This Is An Advanced Article',
        paragraphs: [{
            content: 'This is paragraph one.'
        }, {
            content: 'Here goes the paragraphs sequel'
        }, {
            content: 'Finally the last paragrpah goes here.'
        }]
    }
});
</script>
